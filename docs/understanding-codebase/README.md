## Generation Process

This documentation was generated via a dag-fleet of 5 parallel explorer workers (CLI, Store, DB/Collections, LLM/AST, SDK/MCP) + 1 synthesizer, all running codex gpt-5.5 medium. The fleet analyzed ~15,000 lines of TypeScript source and produced structured findings, which were then synthesized into Q&A format.

Fleet cost: ~$7. Total generation time: ~30 minutes.
